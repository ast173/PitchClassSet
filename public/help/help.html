Help

About Inputs

            To enter a pitch class:
             - Click the square buttons to toggle their corresponding pitch class.

             - Type a note letter using the keyboard.
                - Sharps and flats are supported.
                   - Example: typing 'c' followed by '#' registers as C#.
                - Alternatively, use '+' for sharps and '-' for flats.

             - Type the number corresponding to the pitch class.
                - Example: '0' for 0, '1' for 1, all the way to 't' for 10.

             - Alternatively, enable manual input in settings.
                - A new text field will apear that accepts:
                   1. Pitch classes (numbers or letters)
                   2. Forte numbers

            * Note: 'e' will always refer to the note E, not pitch class 11.

About Outputs

            This program outputs the following information:
             - Normal order
             - Prime form
             - Interval-class vector (IC vector)
             - Complement (unordered)
             - Forte number
             - Z-mate (if it exists)
             - Recognized set
                - Displayed only if the set has a known name, such as a chord or scale.
             - Unordered set (unordered)

About Buttons

            Reset:
             - Clears all square buttons and outputs.
             - Pessing 'ESC' also resets.

            Complement:
             - Gives the complement of the current set.
             - By definition, all square buttons are toggled.

            Random:
             - Generates a random pitch class set.

            Undo:
             - Undoes an action (see "About History").

            Redo:
             - Redoes an action.

            Settings:
             - Switches to the settings page.

            Help:
             - Opens this menu.

About History

            Undo shortcuts:
             - Backspace
             - Delete
             - CTRL + Z
             - Undo button

            Redo shortcuts:
             - CTRL + Y
             - Redo button

            An action includes:
             - Any inputs
             - Reset
             - Complement
             - Random

            * Note: History is cleared after refreshing the page or returning from the settings page.

About Settings

            Notes or letters:
             - Determines whether square buttons show note names or pitch class numbers.

            10/11 or T/E:
             - Independent of the notes or letters setting.
             - Displays pitch classes 10 and 11 as T and E.
             - Affects square buttons and all outputs.

            Type of packing:
             - Rahn: packed from the right.
             - Forte: packed to the left.
             - Only affects 6 total prime forms:
                - 5-20, 6-Z29, 6-31, 7-Z18, 7-20, 8-26

            For Tₙ and TₙI, one or all:
             - Displays either a single transposition/inversion or all 12 simultaneously.
            
            Show Unique Inversions?
             - Currently has no effect.
            
            Manual input on?
             - Displays a text field for direct input.
