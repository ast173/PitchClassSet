Help

About Inputs

            To enter a pitch class (PC):
             - Click the check boxes to toggle their coresponding pc
             - Type the note letter (this also supports sharps and flats) (eg: type 'c' and then '#' for the note C#)
             - Type the number coresponding to the pc (eg: '0' for 0, '1' for 1, all the way to 't' for 10) (Note 'e' will always enter the note E and not the pc 11)

About Outputs

            Outputs:
             - Normal Order
             - Prime form
             - IC Vector
             - Complement (In no particular order)
             - Forte Number
             - Z Mate (if it exists)

About Buttons

            Reset: Clears all checkboxes and outputs
            Complement: Gives the complement (when all checkboxes are toggled)
            Random: Gives a random set

About Transposition (Tₙ) and Inversion (TₙI)

            T0 and T0I are sorted, all other n values are based on n = 0

Other

            Press Backspace, Delete, or CTRL + Z to undo
            Press CTRL + Y to redo
